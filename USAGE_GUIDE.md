# MW Coach Chatbot - User Guide

## Quick Start

### Accessing the Chatbot
1. Open the application in your web browser
2. Look for the purple chat button in the bottom-right corner
3. Click the button to open the chat window

### Main Interface

#### Header Section
- **Bot Avatar**: Visual indicator of the MW Coach assistant
- **Title**: "MW Coach - LMX Platform Assistant"
- **Close Button**: Click to minimize the chat window

#### Category Pills
Horizontal scrollable bar with quick access to topics:
- Platform Overview
- Login and Access
- Billboard Management
- Pricing and Rates
- Tags Management
- Platform Features

Click any category to see all related FAQs.

#### Chat Messages Area
- **Bot Messages**: White bubbles on the left with purple text
- **Your Messages**: Purple gradient bubbles on the right
- **FAQ Results**: Expandable cards with questions and detailed answers
- **Timestamps**: Shows when each message was sent

#### Input Area
- **Text Input**: Type your question or message
- **Send Button**: Purple gradient button to submit your message
- **Keyboard Shortcut**: Press Enter to send

## How to Get Help

### Method 1: Ask Questions Naturally
Simply type your question as you would ask a human:

**Examples:**
- "How do I log in?"
- "What is a programmatic rate?"
- "How can I filter my billboards?"
- "Tell me about tags"

### Method 2: Browse by Category
1. Click on a category pill at the top
2. View all FAQs in that category
3. Read through the answers

### Method 3: Use Keywords
Type keywords related to your question:
- "login"
- "price"
- "tag"
- "billboard"
- "rate"

## Tips for Best Results

### Be Specific
❌ "Tell me about prices"
✅ "How do I change the monthly rate for a billboard?"

### Use Platform Terms
Include terms like:
- Billboard, DOOH, OOH
- Traditional rate, Programmatic rate
- Tags, Registry
- Dashboard, Earnings

### Try Different Phrasings
If you don't get the answer you need:
1. Rephrase your question
2. Try using keywords
3. Browse the relevant category

### Read Full Answers
FAQ answers contain detailed step-by-step instructions. Take time to read through them completely.

## Understanding Responses

### Single FAQ Result
When the bot finds one highly relevant answer:
- Shows the question
- Provides detailed answer
- Displays the category tag

### Multiple Results
When several FAQs match your query:
- Lists all relevant questions
- Shows complete answers for each
- Organized by relevance

### No Results Found
If no matches are found:
- Bot shows the main menu
- Suggests browsing categories
- Offers help options

## Features

### Real-Time Typing Indicator
Three animated dots show when the bot is "thinking" and preparing a response.

### Scroll to Bottom
Chat automatically scrolls to show the latest message.

### Category Tags
Each FAQ is tagged with its category for easy reference.

### Responsive Design
Works on:
- Desktop computers
- Tablets
- Mobile phones

## Keyboard Shortcuts

- **Enter**: Send message
- **Shift + Enter**: New line (if supported)

## Best Practices

1. **Start Broad, Then Narrow**: Begin with general questions, then ask for specifics
2. **Use the Categories**: Browse categories when exploring new topics
3. **Check Related FAQs**: Often multiple FAQs answer different aspects of the same topic
4. **Save Important Info**: Note down steps from detailed instructions

## Getting More Help

If the chatbot doesn't have the answer you need:
1. Try rephrasing your question
2. Browse related categories
3. Look for similar questions in the FAQ list
4. Contact your system administrator or LMX support team

## Technical Notes

- The chatbot uses natural language matching to find relevant FAQs
- Searches across questions, answers, and categories
- No login required to use the chatbot
- All data is from official LMX training materials
